# 🎓 Fode (fode.io)

Welcome to the central repository for **Fode**, an innovative EdTech platform designed to help kids learn code. This repository houses the core codebase that powers [fode.io](https://fode.io).

## 🛠️ Tech Stack
* **Frontend:** React, Tailwind CSS, HTML/CSS/JS
* **Backend:** Node.js, Python
* **Database:** SQL

---

## 📁 Project Structure
* `/frontend` - The user interface and course viewer.
* `/backend` - API services for user data and course delivery.
* `/courses` - Text-based files that dynamically populate [fode.io/courses](https://fode.io/courses).
